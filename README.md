# 🎬 Netflix Clone – React Native App

A mobile application featuring a Netflix-like UI with category-based movie listing, detail screens, search functionality, and state management powered by Redux Toolkit.

## 🚀 Features

🔥 React Native 0.82 — New architecture compatible

🧭 Multi-screen structure with React Navigation

🗂️ Redux Toolkit for global state management

🎭 Movie data fetched from TMDB API

🏷️ Browse movies by category

🔍 Search screen

📄 Movie detail screen

🎨 Modern UI inspired by Netflix

⚡ Fast performance & clean architecture
